Información General OVI

Introducción, profundización e información complementaria sobre el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, HTML5 y CSS3 para creación de contenidos en formato web (Paginas y Sitios Web)

Objetivos del OVI

  • Aplicar los conocimientos de HTML5 y CSS3 para creación de contenidos en formato web (Paginas y Sitios Web)

  • Brindar información a los visitantes sobre el uso de GIT, GITHUB, HTML5 y CSS3

  • Instalación de GIT, GITHUB

Objetivos del Curso y Tematica

Desarrollar sitios web en formato HTML5 y CSS3, para ser visualizados en diferentes navegadores teniendo en cuenta su funcionabilidad y usabilidad, por medio del uso de diferentes editores de código propuestos en el desarrollo del curso.
